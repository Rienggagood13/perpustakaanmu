# Fitur Requirements :
<ol>
<li>Ada 2 jenis User yaitu Admin dan User</li>
<li>Setiap User Harus Melakukan Login Untuk Dapat Mengakses Web</li>
<li>user harus terdaftar sebagai User untuk meminjam buku</li>
<li>Satu user hanya dapat memiliki satu profile</li>
<li>setiap user dapat mengubah profilenya masing"</li>
<li>Setiap Buku dapat memiliki multiple kategori</li>
<li>Judul buku dapat berjumlah lebih dari 1 (dapat dibedakan melalui kode buku)</li>
<li>Bisa melakukan pencaharian buku melalui judul buku</li>
<li>Admin bisa menambah,mengupdate,dan menghapus: data buku,kategori dan user</li>
<li>Setiap User hanya dapat meminjam maksimal 3 buku</li>
<li>Peminjaman maksimal 7 hari. Jika peminjaman lebih dari 7 hari maka akan dikenakan denda</li>
<li>Hanya Admin yang dapat melakukan pengembalian buku, jadi setiap User harus menghubungi admin jika ingin mengembalikan buku.</li>
<li>Admin dapat melihat list buku yang dipinjam, sedangkan User hanya dapat melihat list buku yang dipinjam olehnya</li>
<li>Admin dapat melihat dan mencetak riwayat peminjaman buku, sedangkan User hanya dapat melihat list buku yang dipinjam olehnya</li>
</ol>
